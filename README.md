PROJECT NAME
Personal Portfolio Website

PROJECT DESCRIPTION
This project is a personal portfolio website developed to showcase skills, projects, education details, and contact information in a professional manner. It serves as an online resume and personal branding platform.

Overview
This is a responsive personal portfolio website built using modern web technologies. It displays personal information, skills, completed projects, and contact details in a clean and user-friendly layout.

Features
Responsive design for all devices
About Me section
Skills and projects showcase
Education and achievements section
Contact form / contact details
Simple and modern user interface

Technologies Used
Frontend: HTML, CSS, JavaScript
Styling: CSS Flexbox / Grid
Hosting: GitHub Pages

Project Status
Completed Successfully

Use Case
This project is suitable for internship submission, personal branding, resume linking, and portfolio presentation.
